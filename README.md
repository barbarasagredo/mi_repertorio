# 🎵 Mi Repertorio

Aplicación web simple para la gestión de un repertorio de canciones. Permite visualizar, agregar y modificar canciones en un listado personal.

## ✨ Características

- **Visualizar repertorio:** Muestra una lista de las canciones guardadas con su artista y tono.
- **Agregar canciones:** Permite incorporar nuevas canciones al repertorio a través de un formulario.
- **[Funcionalidad a completar]:** Editar o eliminar canciones (sugerido por la lógica backend).
- **Interfaz limpia:** Diseño simple y funcional construido con HTML, CSS y JavaScript.

## 🛠️ Tecnologías Utilizadas

- **Frontend:** HTML, CSS, JavaScript.
- **Backend:** Node.js con Express.
- **Base de datos:** Archivo JSON (`repertorio.json`) para almacenar los datos.

## 🚀 Cómo Empezar

Sigue estos pasos para poner el proyecto en funcionamiento en tu entorno local.

### Pre-requisitos

- Tener instalado [Node.js](https://nodejs.org/) (que incluye npm).

### Instalación

1.  **Clona el repositorio**
    `git clone https://github.com/barbarasagredo/mi_repertorio.git `
2.  **Navega al directorio del proyecto**
    `cd mi_repertorio `

3.  **Instala las dependencias**
    `npm install`

4.  **Inicia el servidor**
    `node index.js`

5.  Abre tu navegador web y ve a la dirección http://localhost:[PUERTO], donde [PUERTO] es el puerto configurado en tu archivo index.js (comúnmente 3000).
